Admissions Coding Challenge

Getting Started

Please write your solution to the coding challenge in repl.it. repl.it is an online coding environment where you can write code and share the link to your code with other people. Here's how to use it:

  1. Create an account on repl.it by clicking the Sign Up button
  2. Click the red circle in the bottom right hand corner
  3. In the search box type Nodejs
  4. Click on the Nodejs link. You should see a screen like this:
  5. Add your code to the white box on the left. When you click the run button, the results will show up in the black box on the right
  6. Once you've completed the coding challenge, click the share button and copy the link in the Share Link box. Send this link to Steph.

The Challenge

Your task is to create a function called getWeather that will have 2 parameters. One called country and one called weatherType. Call the getWeather function with 2 arguments. The first should be "Scotland", the second should be "sunny". Your function should return the String "The weather in Scotland is sunny". Wrap the call to getWeather in a console.log() to print out the String. Call the getWeather function two more times with countries and weatherTypes of your choice. Remember to wrap your function calls in console.logs!

HINTS:

  • Start by writing a function using the keyword const and the name getWeather
  • Then set the variable = to a set of parentheses followed by an arrow () => and then two curly brackets {}
  • Call the getWeather() function on the last line
